<b>Aplicação FullStack de um sistema de Posts!</b>

Desenvolvimento de uma aplicação Fullstack utilizando Ruby on Rails para o backend e Vue.js com Vite para o frontend. 
O foco principal do projeto foi a integração entre backend, frontend e banco de dados, sem a necessidade de um sistema de usuários, tornando a aplicação mais simples e direta.

Essa aplicação demonstra uma estrutura fullstack funcional e enxuta, permitindo integração entre backend, frontend e banco de dados de maneira prática e eficiente. Foi projetada para ser simples, porém escalável, servindo como base para futuras melhorias e expansões.

<b>Backend: Ruby on Rails + PostgreSQL</b>

No backend, utilizei Ruby on Rails para criar uma API RESTful que gerencia os posts. A aplicação permite operações CRUD (Create, Read, Update, Delete) para manipular os dados dos posts, armazenando as informações em um banco PostgreSQL.
A API foi estruturada para responder de forma eficiente às requisições do frontend, retornando os dados em JSON.

<b> Frontend: Vue.js + Vite </b>

Para o frontend, escolhi Vue.js por sua reatividade e facilidade de integração com APIs. Utilizei Vite como ferramenta de build, garantindo um ambiente de desenvolvimento rápido e otimizado.
A interface permite visualizar uma lista de posts, criar novos posts, editar e excluir posts existentes. Todas as interações com o backend acontecem de forma assíncrona, proporcionando uma experiência fluida para o usuário.

<b>Funcionalidades:</b>
- Integração Backend + Frontend: Comunicação fluida entre Rails e Vue.js via API RESTful.
- Banco de Dados PostgreSQL: Armazenamento estruturado e eficiente dos posts.
- CRUD Simples: Funcionalidades essenciais para gerenciar posts.
- Frontend Responsivo: Interface leve e interativa utilizando Vue.js.
